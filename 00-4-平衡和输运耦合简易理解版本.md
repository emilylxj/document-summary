托卡马克里“平衡”和“输运”迭代，本质上是在解两个相互耦合的问题：

1. 磁平衡决定磁面长什么样
2. 输运决定等离子体在这些磁面上如何演化

两者互相依赖，所以通常要迭代。

**一、平衡在解什么**  
平衡主要是指 MHD 力平衡：

J×B=∇pJ×B=∇p

在轴对称托卡马克里，它通常被写成 Grad-Shafranov 方程：

Δ∗ψ=−μ0R2p′(ψ)−F(ψ)F′(ψ)Δ∗ψ=−μ0​R2p′(ψ)−F(ψ)F′(ψ)

这里：

- ψψ 是极向磁通，决定磁面
- p′(ψ)p′(ψ) 是压强对磁通的导数
- F(ψ)=RBϕF(ψ)=RBϕ​
- FF′(ψ)FF′(ψ) 反映环向场函数对应的电流贡献

所以平衡求解器需要的核心输入，不是温度密度本身，而是更接近：

- p(ψ)p(ψ) 或 p′(ψ)p′(ψ)
- F(ψ)F(ψ) 或 FF′(ψ)FF′(ψ)
- 总电流、边界形状、线圈条件等

平衡解出来以后，会得到：

- 磁面几何
- q(ψ)q(ψ)
- 磁轴位置
- LCFS
- 磁面体积、面积、Rin/outRin/out​、伸长率、三角形度等几何量

这些都是输运求解所需要的“几何背景”。

**二、输运在解什么**  
输运是在每个磁面上解守恒方程，典型包括：

- 粒子输运
- 电子能量输运
- 离子能量输运
- 电流扩散

形式上像一组 1D 径向方程，例如：

∂n∂t=−1V′∂∂ρ(V′Γn)+Sn∂t∂n​=−V′1​∂ρ∂​(V′Γn​)+Sn​

∂We∂t=−1V′∂∂ρ(V′qe)+Pe∂t∂We​​=−V′1​∂ρ∂​(V′qe​)+Pe​

∂Wi∂t=−1V′∂∂ρ(V′qi)+Pi∂t∂Wi​​=−V′1​∂ρ∂​(V′qi​)+Pi​

这里：

- ρρ 是径向磁面坐标
- V′V′ 是磁面微分体积
- ΓnΓn​ 是粒子通量
- qe,qiqe​,qi​ 是热流
- Sn,Pe,PiSn​,Pe​,Pi​ 是源项

输运系数一般又依赖于：

- 温度、密度梯度
- 磁剪切、qq
- 磁面几何
- 电流分布
- 湍流模型或经验约束

所以输运不是脱离平衡单独求的，它依赖平衡给出的磁面几何和磁拓扑。

**三、为什么要迭代**  
因为两边是闭环耦合的。

平衡需要：

- p′(ψ)p′(ψ)
- FF′(ψ)FF′(ψ)
- 电流分布

这些量来自输运或电流演化结果。

输运需要：

- 磁面几何
- V′V′
- 面积因子
- qq
- 磁剪切
- 有时还需要 trapped fraction、连接长度、平均磁场等

这些量来自平衡解。

所以逻辑上是：

1. 先猜一个平衡
2. 用这个平衡几何去解输运
3. 输运更新压力、电流、bootstrap、电阻扩散结果
4. 把新的 p′、FF′p′、FF′ 或电流约束送回平衡求解器
5. 再解一个新平衡
6. 循环直到收敛

**四、最常见的两种迭代模式**

**1. 静态自洽迭代**  
目标是求一个稳态或准稳态解。

流程通常是：

1. 给定边界和总电流
2. 猜测 p(ψ),j(ψ)p(ψ),j(ψ) 或 q(ψ)q(ψ)
3. 解 Grad-Shafranov
4. 提取几何量
5. 解稳态输运方程
6. 得到新的压强、电流、bootstrap
7. 重新构造 p′(ψ),FF′(ψ)p′(ψ),FF′(ψ)
8. 回到平衡
9. 直到几何、压强、电流变化足够小

这是你现在讨论的 CHEASE 和 TORAX 耦合更接近的模式。

**2. 时间推进中的内迭代**  
目标是做时域演化。

在每个时间步 tn→tn+1tn→tn+1：

1. 用上一步平衡作为初值
2. 先推进输运方程
3. 得到新的 Te,Ti,ne,jTe​,Ti​,ne​,j
4. 更新 pressure 和 current profile
5. 重算平衡
6. 用新平衡修正输运系数和几何项
7. 必要时做 Picard / Newton 内迭代
8. 时间步收敛后进入下一步

这类算法里，“时间步外层 + 平衡输运内层迭代”很常见。

**五、物理上谁驱动谁**  
更准确地说，不是“平衡驱动输运”或者“输运驱动平衡”单向成立，而是：

- 平衡提供磁面和几何约束
- 输运提供 profile 演化和源汇后的压力电流结果
- 二者共同决定下一轮状态

但如果分角色讲：

- 平衡更像“约束器”
- 输运更像“演化器”

**六、在实际代码里怎么耦合**  
你现在这个问题，最核心的接口通常就是：

从输运到平衡：

- p′(ψ)p′(ψ)
- FF′(ψ)FF′(ψ)
- 或者 pressure + current constraints

从平衡到输运：

- 磁面几何
- V′(ρ)V′(ρ)
- 面积、周长、雅可比
- Rin,RoutRin​,Rout​
- elongation、triangularity
- q(ρ)q(ρ)、磁剪切
- 可能还有更高阶 metric

这也是为什么你之前强调“几何比 q 更该先检查”是有道理的。对输运来说，首先必须保证磁面几何和坐标映射是对的，否则后面的体积因子、梯度算子、源项投影都会错。

**七、收敛判据一般看什么**  
一个平衡-输运闭环是否收敛，常看：

- 压力 profile 是否稳定
- 电流 profile 是否稳定
- p′(ψ)p′(ψ)、FF′(ψ)FF′(ψ) 是否稳定
- 磁轴位置、LCFS、几何量是否稳定
- q(ρ)q(ρ) 是否稳定
- 功率平衡残差是否小
- 相邻两次迭代变化是否足够小

比如可以看：

∥pk+1−pk∥,∥qk+1−qk∥,∥ψk+1−ψk∥∥pk+1−pk∥,∥qk+1−qk∥,∥ψk+1−ψk∥

**八、为什么这个问题很容易出错**  
因为耦合点很多：

- 坐标不一致：ψ,ρ,ρtor,Φψ,ρ,ρtor​,Φ
- 符号不一致：COCOS、p′p′, FF′FF′, 电流方向
- 几何量定义不一致：Rin/Rout、三角形度定义、磁面平均方式
- 一个代码给的是“约束量”，另一个代码需要的是“导出量”
- 一个代码默认固定电流，另一个默认演化电流

所以工程上常见策略不是一上来做闭环，而是：

1. 先单向平衡 →→ 输运打通
2. 再检查几何一致性
3. 再检查 sign / COCOS
4. 再检查 profile consistency
5. 最后再闭环

这和你们现在的推进顺序是一致的。

**九、如果用一句话概括**  
托卡马克平衡-输运迭代的原理就是：

“用平衡解给输运提供磁面几何，用输运解给平衡提供压力和电流约束，在两者之间反复更新，直到磁结构和等离子体 profile 自洽。”
